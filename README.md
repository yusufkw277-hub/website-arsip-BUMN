# Website Arsip BUMN

Website Arsip BUMN adalah aplikasi berbasis web yang digunakan untuk mengelola arsip inaktif secara digital.  
Aplikasi ini dibuat untuk membantu proses penyimpanan, pencarian, dan pengelolaan data arsip agar lebih terstruktur dan mudah diakses.
